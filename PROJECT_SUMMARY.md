# סיכום פרויקט - מערכת חילוץ נתונים ממסמכים

## 📋 תיאור הפרויקט

מערכת מתקדמת לחילוץ נתונים אוטומטי ממסמכים בעברית ובאנגלית. המערכת מאפשרת העלאת מסמכים שונים (Word, PDF, טקסט) וחילוץ אוטומטי של מידע רלוונטי כמו שמות חברות, סכומים, תאריכים, פרטי קשר ועוד.

## ✅ מה הושלם

- ✅ קונפיגורציה מלאה של כלי הבנייה

### 🎨 ממשק המשתמש

## 🛠️ טכנולוגיות בשימוש

- **React 18** - ספריית JavaScript מודרנית
- **Tailwind CSS** - מסגרת CSS עם תמיכה ב-RTL
- **Lucide React** - אייקונים מודרניים
- **XLSX** - עבודה עם קבצי Excel
- **Mammoth** - חילוץ טקסט מקבצי Word
- **JavaScript RegExp** - ביטויים רגולריים מתקדמים

## 🚀 הוראות הפעלה

### הפעלה מהירה
```bash
chmod +x run.sh
./run.sh
```

### פרסום
```bash
chmod +x publish.sh
./publish.sh
```

### הפעלה ידנית
```bash
npm install
npm start        # לפיתוח
npm run build    # לפרסום
```
