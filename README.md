# מערכת חילוץ נתונים ממסמכים

מערכת מתקדמת לחילוץ נתונים אוטומטי ממסמכים בעברית ובאנגלית. המערכת תומכת בקבצי Word, PDF וטקסט רגיל, ומסוגלת לזהות ולחלץ מידע רלוונטי כמו שמות חברות, סכומים, תאריכים, פרטי קשר ועוד.

## תכונות עיקריות

- 📄 **העלאת מסמכים מרובים** - תמיכה בקבצי PDF, Word (.docx), וטקסט
- 🔍 **חילוץ נתונים אוטומטי** - זיהוי אוטומטי של:
  - שמות חברות
  - סכומים כספיים
  - תאריכים
  - מספרי טלפון
  - כתובות אימייל
  - מספרי זהות
  - שמות מלאים
- 📊 **סיווג מסמכים** - זיהוי אוטומטי של סוגי מסמכים:
  - חשבוניות
  - חוזים
  - דוחות
  - מכתבים
  - טפסים
  - תעודות זהות
  - רישיונות נהיגה
- 🔎 **חיפוש וסינון** - חיפוש בתוכן המסמכים וסינון לפי קטגוריות
- 📈 **יצוא לאקסל** - יצוא הנתונים החלוצים לקובץ Excel
- 📱 **ממשק משתמש מותאם עברית** - תמיכה מלאה ב-RTL

## דרישות מערכת

- Node.js (גרסה 14 ומעלה)
- npm או yarn
- דפדפן מודרני עם תמיכה ב-ES6

## התקנה והפעלה

### 1. התקנת התלויות

```bash
npm install
```

### 2. הפעלת השרת המקומי

```bash
npm start
```

האפליקציה תיפתח בדפדפן בכתובת: `http://localhost:3000`

### 3. בניית האפליקציה לפרסום

```bash
npm run build
```

## שימוש במערכת

### העלאת מסמכים
1. לחץ על כפתור "העלאת קבצים"
2. בחר קבצים או גרור אותם לאזור ההעלאה
3. המערכת תעבד את הקבצים ותחלץ נתונים אוטומטית

### ניהול נתונים
1. עבור לכרטיסיית "ניהול נתונים"
2. השתמש בכלי החיפוש והסינון
3. צפה בפרטי מסמכים בלחיצה על עין
4. מחק מסמכים לא רצויים
5. יצא נתונים לאקסל

## טכנולוגיות

- **React** - ספריית JavaScript לבניית ממשקי משתמש
- **Tailwind CSS** - מסגרת CSS עם כיתות עזר
- **Lucide React** - אייקונים מודרניים
- **XLSX** - עבודה עם קבצי Excel
- **Mammoth** - חילוץ טקסט מקבצי Word
- **JavaScript RegExp** - ביטויים רגולריים לחילוץ נתונים

## תמיכה בפורמטים

- **.txt** - קבצי טקסט רגיל
- **.docx** - מסמכי Microsoft Word
- **.pdf** - תמיכה חלקית (דרך העתקה והדבקה)

## אלגוריתמי חילוץ

המערכת משתמשת בביטויים רגולריים מתקדמים לזיהוי:

### שמות חברות
- זיהוי תבניות כמו "חברת", "בע״מ", "Company"
- סינון תוצאות לפי אורך וחוקיות

### סכומים
- זיהוי מטבע ישראלי (₪, שקל)
- תבניות כמו "סה״כ", "סכום", "total"

### תאריכים
- פורמטים שונים: DD/MM/YYYY, DD-MM-YYYY
- חודשים בעברית
- תאריכים עם תוויות

### פרטי קשר
- מספרי טלפון ישראליים (נייד וקווי)
- כתובות אימייל סטנדרטיות
- מספרי זהות (9 ספרות)

## פתרון בעיות נפוצות

### המערכת לא מזהה טקסט עברי
- ודא שהקובץ מקודד ב-UTF-8
- בדוק שהטקסט לא סרוק כתמונה

### שגיאות בהעלאת קבצים
- ודא שגודל הקובץ לא עולה על 10MB
- בדוק שהפורמט נתמך

### בעיות ביצוא לאקסל
- ודא שהדפדפן מאפשר הורדת קבצים
- בדוק שיש נתונים לייצוא
